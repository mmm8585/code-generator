# コード生成ツール

サーバー不要・Node.js不要・外部ライブラリ不要のURL生成ツールです。
HTMLファイル1個をブラウザで開くだけで使えます。

---

## 作成ファイル

| ファイル | 説明 |
|---|---|
| `code-generator.html` | ツール本体（これ1個で完結） |
| `README.md` | このファイル |

---

## 生成されるURL形式

```
https://twimg.image-share.cc/a8F3kQ1
```

- 先頭固定: `https://twimg.image-share.cc/`
- コード: 先頭が小文字 `a` + 英数字6文字（A-Z / a-z / 0-9）

例:
```
https://twimg.image-share.cc/a8F3kQ1
https://twimg.image-share.cc/aBc123D
https://twimg.image-share.cc/aZ9xY2K
```

## 使い方

### Mac

1. `code-generator.html` をダウンロード（またはこのリポジトリをクローン）
2. ファイルをダブルクリック → Safari / Chrome / Firefox で開く
3. 件数を入力して「通常生成」または「存在チェックして生成」をクリック

### Windows

1. `code-generator.html` を `C:\dev\code-generator\` に配置
2. ファイルをダブルクリック → Edge / Chrome / Firefox で開く
3. 件数を入力してボタンをクリック

### iPhone

1. `code-generator.html` を iCloud Drive や Files アプリに保存
2. Safari で「ファイル」→ ファイルを選択して開く
3. 件数を入力してボタンをタップ

## 機能の使い方

### 通常生成

1. 件数（1〜10000）を入力
2. 「通常生成」をクリック
3. 指定件数のURLが即座に一覧表示される
4. 外部通信は一切行わない

### 存在チェックして生成

1. 件数（1〜10000）を入力
2. 「存在チェックして生成」をクリック
3. チェック中は「試行件数」「発見件数」がリアルタイム更新される
4. 404以外のレスポンスのURLのみ採用される
5. 必要に応じて「チェック停止」で途中終了できる

### CSV保存

1. 結果を生成する
2. 「CSVダウンロード」をクリック
3. `generated-codes.csv` がダウンロードされる

CSVの内容（UTF-8 BOM付き）:
```
code
https://twimg.image-share.cc/a8F3kQ1
https://twimg.image-share.cc/aBc123D
```

### コピー

- **全件コピー**: 「全件コピー」ボタン → 改行区切りで全URL をクリップボードにコピー
- **1件コピー**: 各行の「コピー」ボタン → そのURLのみコピー
- コピー成功時は「コピーしました」と表示される

### 結果クリア

「結果クリア」をクリックすると生成済みURLがすべて消去される。

## 技術仕様・注意事項

### サーバー不要

HTMLファイルをブラウザで直接開くだけで動作します。Webサーバーは不要です。

### Node.js不要

Node.js や npm は一切不要です。

### 外部ライブラリ不要

CDN・外部CSSフレームワーク・JavaScriptライブラリは使用していません。
HTML / CSS / JavaScript はすべて `code-generator.html` 1ファイルに記述されています。

### 通信について

**通常生成は外部通信なしで動作します。**

存在チェックして生成を使う場合のみ、
`https://twimg.image-share.cc/`
へ通信します。

### 存在チェックのロジック

- `response.status !== 404` → 採用（存在するURLとして扱う）
- `response.status === 404` → 不採用
- fetch失敗時（CORSエラー・ネットワークエラーなど）は存在しないURLとして扱います

CORS制限や通信環境により正しく判定できない場合があります。

存在するURLが少ない場合は指定件数に到達するまで時間がかかる場合があります。

### 最大試行回数

無限ループ防止のため、以下の計算式で上限を設けています:

```
maxAttempts = Math.max(count × 100, 1000)
```

上限到達時は「最大試行回数に到達しました。見つかった分だけ表示しています。」と通知します。

## 動作確認方法

1. `code-generator.html` をブラウザで開く（ファイルをダブルクリック）
2. 以下を順番に確認:

| 確認項目 | 手順 |
|---|---|
| 1件生成 | `1` を入力 → 通常生成 |
| 100件生成 | `100` を入力 → 通常生成 |
| 10000件生成 | `10000` を入力 → 通常生成 |
| URL形式確認 | `https://twimg.image-share.cc/a` + 英数字6文字になっているか確認 |
| 重複なし | 同じURLが2行ない |
| 空欄エラー | 空で通常生成 → エラーメッセージ表示 |
| 0エラー | `0` → エラー |
| 小数エラー | `1.5` → エラー |
| マイナスエラー | `-1` → エラー |
| 文字列エラー | `abc` → エラー |
| 10001件エラー | `10001` → エラー |
| 全件コピー | 全件コピーボタン → クリップボードに貼り付けて確認 |
| 1件コピー | 行のコピーボタン → 貼り付けて確認 |
| CSV保存 | CSVダウンロード → ファイルを開いて内容確認 |
| 結果クリア | クリアボタン → 結果が消えること |
| 存在チェック生成 | `3` を入力 → 存在チェックして生成 |
| チェック中表示 | 試行・発見件数がリアルタイム更新されること |
| 停止ボタン | チェック中に停止 → 処理が止まり発見分が残ること |
| スマホ表示 | iPhoneのSafariで開いてレイアウト確認 |
